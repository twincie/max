## Setup
to run this project:
* if you dont have python installed already go to the official website to get it. but if you have it cool.
* install pyqt5 using pip 
```  
$ pip install pyqt5
```
* install mutagen also using pip 
```  
$ pip install mutagen
```
* To then run the app, on the terminal, run
```  
$ py main.py
```
